What browsers does impact support?

Akshay
Follow

If you are using desktop, we recommend using the latest version of Chrome, Firefox, or Edge.

Note: You can absolutely use browsers that we partially support, but things might not always look right. If you’re using a partially supported browser and something is off, please reattempt with a fully supported browser.

Fully supported browsers:

Device Type Browser Version
Desktop Chrome 68 and up
Mobile S afari (in-app) 10.0 and up
Mobile Safari 11 and up
Mobile Chrome 69 and up
Mobile Android Webview 69 and up


Partially supported browsers:

Device Type Browser Version
Desktop Edge 17 and up
Desktop Firefox 63 and up
Mobile Samsung Internet 7 and up
